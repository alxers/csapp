#define M // Mystery number 1
#define N // Mystery number 2

int arith(int x, int y) {
	int result = 0;
	result = x*M + y/N;
	return result;
}

// Translation of assembly code for arith
int optarith(int x, int y) {
	int t = x;
	x <<= 5;
	x -= t;
	if (y < 0) y += 7;
	y >>= 3; // Arithmetic shift
	return x+y;
}

What are the values of M and N?

Example: x = 2, y = 3
x = 62, y = 0, x + y = 62
